一、 试卷结构透视

通过分析近三年的试卷，可以发现试卷结构极度稳定。如果你能熟练掌握以下7类计算题的解法，及格甚至高分都轻而易举。

二、 核心考点与复习路线

第一阶段：必拿分的“七大计算题” (占卷面约60%分数)

这部分是复习的重中之重，建议直接背诵解题步骤模板。

1. 解析性与可导性 (对应第11题)

  • 考点：给定 f(z)=u(x,y)+iv(x,y)，判断哪里可导，哪里解析。
  • 核心公式：柯西-黎曼方程 (C-R方程)
    ∂x∂u​=∂y∂v​,∂y∂u​=−∂x∂v​
  • 解题步骤：
    1. 求偏导 ux​,uy​,vx​,vy​。
    2. 令其满足 C-R 方程，解出 x,y 的关系。
    3. 结论：在满足方程的点可导；如果这些点构成了区域（不仅仅是孤立点或线），则在区域内解析。通常题目结果往往是“仅在某些点可导，处处不解析”或“处处解析

2. 路径积分计算 (对应第12题)

  • 考点：计算 ∫C​f(z)dz，注意 C 通常不是闭合的。
  • 解法判断：
    • 方法A (原函数法)：如果 f(z) 在包含 C 的单连通区域内解析，直接用 F(终点)−F(起点)。例如 ∫ezdz。
    • 方法B (参数法)：如果 f(z) 含 zˉ,Re(z) 等不解析项，必须设 z(t)=x(t)+iy(t)，代入转化为 t 的定积分。

3. 泰勒级数展开 (对应第13题)

  • 考点：在 z0​ 处展开 f(z)。
  • 技巧：
    • 换元法：令 t=z−z0​，将 f(z) 变形为关于 t 的函数。
    • 常用公式：熟记 1−t1​=∑tn,et,sint,cost 的展开式。
    • 注意：一定要写出收敛半径或收敛域。

4. 洛朗级数与奇点 (对应第14题)

  • 考点：在指定圆环域（如 0<∣z∣<∞）展开，并判断 z=0 是什么奇点。
  • 核心逻辑：
    • 利用级数展开公式（如 sinz1​=z1​−3z31​+…）。
    • 奇点判别：
      • 无负幂项 → 可去奇点。
      • 有限个负幂项 (最高为 1/zm) → m 级极点。
      • 无穷多个负幂项 → 本性奇点。

5. 留数定理求积分 (对应第15题)

  • 考点：∮C​f(z)dz，其中 C 内部包含若干奇点。
  • 步骤：
    1. 找出 C 内部的所有奇点 zk​。
    2. 计算每个奇点的留数 Res[f(z),zk​]。
      • 一级极点：limz→z0​​(z−z0​)f(z)。
      • 高阶极点：用导数公式。
    3. 结果 =2πi∑Res。

6. 傅里叶变换 (对应第16题)

  • 考点：求 F(ω)，题目通常涉及 δ(t)。
  • 核心记忆：
    • F[δ(t)]=1 (或根据定义不同可能有常数差异，需看教材定义)。
    • F[1]=2πδ(ω)。
    • 位移性质：δ(t−t0​)↔e−iωt0​。
    • 线性性质。

7. 拉普拉斯变换解ODE (对应第17题)

  • 考点：解 y′′+ay′+by=f(t)，给定初值。
  • 步骤：
    1. 两边取拉氏变换，利用 L[y′]=sY−y(0), L[y′′]=s2Y−sy(0)−y′(0)。
    2. 代入初值，解出 Y(s)。
    3. 对 Y(s) 进行部分分式分解。
    4. 取逆变换 L−1 得到 y(t)。

第二阶段：概念巩固 (选择填空)

  • 辐角主值：Arg(z) 范围是 (−π,π]。注意象限，例如 z=1+3​i 在第一象限，角度 π/3；z=−1−i 在第三象限，角度 −3π/4。
  • 收敛半径 R：
    • 比值法 lim∣cn​/cn+1​∣。
    • 距离法：收敛半径 = 展开中心到最近奇点的距离（非常管用！）。
  • 映射 w=1/z：
    • 圆过原点 → 变为直线。
    • 圆不过原点 → 变为圆。
    • 直线过原点 → 变为直线。
  • 调和函数：已知 u 求 v。利用 dv=vx​dx+vy​dy=−uy​dx+ux​dy，然后积分。

三、 考前预测与模拟建议

2025年期末考题预测

根据近三年趋势，今年的试卷极大概率依然是：

  1. 选择/填空：会考一个 i 的高次幂或复数开方；会考一个收敛半径计算；会考一个简单的留数计算。
  2. 大题11：给一个 x,y 的多项式函数，问哪里可导。
  3. 大题12：考一条直线段或圆弧上的积分，函数可能是多项式或指数函数。
  4. 大题13：泰勒展开，分母可能是 (z−a)(z−b) 形式，需要拆项。
  5. 大题14：洛朗展开，重点复习 e1/z, sin(1/z), cos(1/z) 这种类型，判断奇点。
  6. 大题15：分母有高次幂的积分，例如 (z−1)2(z−2)1​，注意只计算圆内的奇点留数。
  7. 大题16：含 δ 函数的傅里叶变换。
  8. 大题17：二阶常系数微分方程，右边是 eat 或 sin/cos。

学习建议

  1. 刷真题：把这份文档里的三年真题的大题部分，盖住答案完完整整做一遍。这是最高效的。
  2. 背公式：
    • C-R方程。
    • 高阶极点留数公式：(m−1)1​limz→z0​​dzm−1dm−1​[(z−z0​)mf(z)]。
    • 拉普拉斯变换的导数性质。
  3. 注意书写规范：复变函数很多题目计算量不大，但逻辑要求严密（如“在…区域内解析”），步骤分很重要。
