我们可以这样理解短路运算符
- 短路运算符一般是用于判断的二元运算符
- 当二元运算符的前项可以直接决定整个表达式的结果时候，二元运算符就会直接**忽略**后项
```cpp
// 在c++中， && || 是短路运算符
int check = 0;
if( 0 && check++ )
{
	//code
}
cout << check << ' ';
```
这时候 check 的输出会是 0 
```cpp
// 对 || 而言
int check = 1;
if( 1 || check-- )
{
	// code
}
cout << check << ' ';
```
这时候 check 的输出会是 1
